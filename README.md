O Game Vault é um site desenvolvido com o objetivo de simular uma loja virtual de jogos digitais, focada na venda de chaves de ativação (keys) da Steam. O projeto foi construído utilizando HTML e CSS, apresentando uma página inicial com listagem de produtos, páginas individuais para cada jogo com descrição, imagem e preço, além de um sistema básico de navegação com páginas de login e cadastro. O principal objetivo do projeto é aplicar conceitos de desenvolvimento web e organização de estruturas de um site real de forma simples e funcional.
